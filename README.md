# Verificador de idade e nome com pergunta de Torta preferida

### Neste repositório há um arquivo em python que:
- Pergunta seu nome e idade.
- Verifica se você tem a idade igual o maior 18.
- Se sim te dá 3 opções de torta preferida.
- Se não retorna que você não tem permissão para acessar.
